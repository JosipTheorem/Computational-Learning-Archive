# Computational Learning Archive

This repository collects older learning projects, notebooks, and experiments in
computational physics, mathematics, numerical methods, and machine learning.

The original repositories were imported into subfolders so the work can be kept
together while preserving the old project histories.

## Contents

- `physics-simulations/` - simulations of physical systems in Python, including
  mechanics, optics, electrodynamics, orbital mechanics, graphene, and Monte
  Carlo experiments.
- `mathematical-sets-and-sequences/` - Python explorations of mathematical
  sequences and visual patterns.
- `numerical-methods/` - notebooks and code for interpolation, integration,
  differentiation, equations, LU decomposition, and differential equations.
- `machine-learning/` - introductory traditional machine learning and deep
  learning notebooks.
- `islp-book-problems/` - selected solutions from *An Introduction to
  Statistical Learning with Python*.

## Notes

This is an archive-style repository rather than a single polished package. Some
folders may have their own README files, dependencies, data assumptions, or
notebook outputs from the original projects.
