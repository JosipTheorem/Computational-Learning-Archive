# ISLP Book Problems

This folder contains selected solutions and experiments from *An Introduction to
Statistical Learning with Python*.

The work is organized by book chapter, with most exercises written as Jupyter
notebooks. The notebooks combine code, plots, short explanations, and exploratory
checks, so they are meant to be read interactively rather than used as a Python
package.

## Contents

- `Chapter 2/` - introductory Python, data exploration, and basic plotting
  exercises.
- `Chapter 3/` - regression-related exercises using `statsmodels` and
  `scikit-learn`.
- `Chapter 4/` - classification examples, including logistic regression, LDA,
  QDA, naive Bayes, and work with the Boston dataset.
- `Chapter 5/` - resampling and validation exercises.
- `Chapter 6/` - model selection and regularization exercises.
- `Chapter 8/` - tree-based methods and related exercises.
- `Chapter 9/` - support vector machine exercises.
- `Chapter 10/` - neural network examples, including fully connected networks on
  the Default dataset and an RNN on the NYSE dataset.
- `Chapter 12/` - selected unsupervised learning exercises.

## Running The Notebooks

These notebooks were written against the Python ecosystem used in the ISLP book.
For setup instructions, datasets, helper functions, and package references, use
the official ISLP resources:

- [ISLP documentation](https://intro-stat-learning.github.io/ISLP/)
- [ISLP package repository](https://github.com/intro-stat-learning/ISLP)
- [Official ISLP labs repository](https://github.com/intro-stat-learning/ISLP_labs)

The deep learning notebooks in `Chapter 10/` may also require the PyTorch-related
packages used by the official ISLP deep learning labs.
